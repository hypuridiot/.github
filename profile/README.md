# hypuridiot

On-chain HIP-3 vaults on Hyperliquid: a tokenized vault that trades HIP-3
perps via CoreWriter, plus a platform for managers to run their own.

| Repo | What it is |
|------|------------|
| [contracts](https://github.com/hypuridiot/contracts) | Solidity (Foundry): Vault, Gateway, RiskGuard, NavOracle |
| [backend](https://github.com/hypuridiot/backend) | Go: indexer, NAV, keeper/sweeper, REST API |
| [frontend](https://github.com/hypuridiot/frontend) | React: depositor app + manager terminal |
| [docs](https://github.com/hypuridiot/docs) | Architecture, ADRs, runbooks |
